About oasp4j-modules

Reusable modules of the Open Application Standard Platform for Java (OASP4J).

Project Modules

This project has declared the following modules:

Name Description
oasp4j-logging Logging Module of the Open Application Standard Platform for Java (OASP4J).
oasp4j-test Module with code and configuration for tests of the Open Application Standard Platform for Java (OASP4J).
oasp4j-configuration Configuration Module of the Open Application Standard Platform for Java (OASP4J).
oasp4j-beanmapping Minimal shim for bean mapping to convert between compatible Java beans (e.g. JPA entity to transfer-object and vice versa).
oasp4j-service Service Support Module of the Open Application Standard Platform for Java (OASP4J).
oasp4j-json JSON Support Module of the Open Application Standard Platform for Java (OASP4J).
oasp4j-rest REST-Service Support Module of the Open Application Standard Platform for Java (OASP4J).
oasp4j-cxf-client Basic support for consuming services as client based on Apache CXF.
oasp4j-cxf-client-rest Support for consuming REST services as client based on Apache CXF.
oasp4j-cxf-client-ws Support for consuming web-service (SOAP) as client based on Apache CXF.
oasp4j-cxf-server Basic support for providing services as server based on Apache CXF.
oasp4j-cxf-server-rest Support for providing REST services on as server based on Apache CXF.
oasp4j-cxf-server-ws Support for providing web-services (SOAP) as server based on Apache CXF.
oasp4j-security Security Module of the Open Application Standard Platform for Java (OASP4J).
oasp4j-jpa JPA-based persistence infrastructure of the Open Application Standard Platform for Java (OASP4J).
oasp4j-jpa-envers JPA-based persistence infrastructure of the Open Application Standard Platform for Java (OASP4J).
oasp4j-batch Batch infrastructure of the Open Application Standard Platform for Java (OASP4J).
oasp4j-web Module for reusable web and servlet related code of the Open Application Standard Platform for Java (OASP4J).
oasp4j-basic Basic code for common usage (such as base classes for transfer objects) of the Open Application Standard Platform for Java (OASP4J).
