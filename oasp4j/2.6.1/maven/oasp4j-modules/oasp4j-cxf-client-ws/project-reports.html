Generated Reports

This document provides an overview of the various reports that are automatically generated by Maven . Each report is briefly described below.

Overview

Document Description
Source Xref HTML based, cross-reference version of Java source code.
Javadoc Javadoc API documentation.
Tag List Report on various tags found in the code.
dependency-check Generates a report providing details on any published vulnerabilities within project dependencies. This report is a best effort and may contain false positives and false negatives.
Service Documentation Documentation of your Services from JAX-RS annotated code and JavaDoc-Comments.
Third Parties Provides details of third parties of the project.
