OASP 4 Java!

The Open Application Standard Platform for Java (OASP4j) provides a standardized architecture blueprint, an open best-of-breed technology stack as well as industry proven best practices and code conventions.

Got stuck with finding the right hammer for your screws? Stop with integration experiments, with digging in the docs of dozens different open-source projects in parallel getting on the wrong track or lost in space.

Start concentrating on the things you need to be productive and to build your solution efficiently and successfully.

From developers for developers!
